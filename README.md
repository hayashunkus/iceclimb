2025工大祭企画

ICE Ascent with hands

手の骨格検知で手をグーパーすると、キャラクターがスピードクライミングをして、リズムよくグーパーすると登っていく。100ｍ登ると登頂成功

各種ゲーム内容

【通常モード】
100mのコースになっていて、紫のひし形のホールドをつかみながら上に登っていく。
PCのカメラに手を向ける。「パー」にして手を移動させて「グー」でホールドをつかむ。
中指でデコピンをすると敵を倒すことができる。
敵はゆっくり落ちてくるのでデコピンで倒しながら上を目指して登っていく。
敵が画面下に到達してしまうとゲームオーバーになってしまう。
敵を５回倒すとブーストして１０ｍ上昇（ワープ）することができる。
手は画面左下の骨格検知画面に表示されるので、カメラの範囲内で手を動かす。
100mのゴール地点には大きい六角形のホールドがあるため、それを両手でつかむとゴールすることができる。
（Rキーで90ｍ地点までワープが可能）
＋αのマウンテンアイスクライマーは工大祭特別版で2人ペアで一人がゲーム操作中にもう片方はマウンテンクライマー（筋トレ）をして、マウンテンクライマーがギブアップしたらそこで終了。100m登り切ったら敬意を表する。（体育会系限定企画）

【タイムアタック】
50mのコースでスピードタイムアタック通常モードよりホールドの間隔が狭く設計してあるため、テンポよく登るのがコツ
敵はいない。

【１分間レース】
１分間でどれだけ登れるかをチャレンジする。
これもタイムアタック同様、ホールドの間隔は狭い。
敵はいない。

【デコピンチャレンジ】
1分間にデコピン（中指ではじく）で何体の敵を倒すことができるかのチャレンジ。
敵は毎秒４体生成され、画面４つ角からランダムに出現する。紫色の敵はデコピン１回で倒せるが、赤色の敵はなかなかしぶとい（３回のデコピンで倒せる）。オレンジ色の敵は５回デコピンしないと倒せない。
赤色の敵は紫色の敵が10体生成されるごとに出現する。オレンジ色の敵は赤色の敵が５体生成されるごとに出現する。
画面上に敵が埋まってしまうとゲームオーバーになってしまう。（画面上に40体ぐらい）

【ルーラーキャッチ】
反射神経のゲームで上から落ちてくる物体を手をパーからグーにすることで、つかむことができたら成功、つかむことができず物体が画面下に消えていったら失敗。

【必殺技を打とう】→後で修正する。
手をグーにして前に出すとパンチが打てる。手をパーにして前に出すと気弾が放てる（片手）。両手をパーにして突き出すと波動が打てる。
敵のHPを削って敵を撃破しよう。
【格闘ゲームロジック】
【位置】
kihon.pngをプレイ画面左側中央、damager.pngがプレイ画面右側中央に配置する。
【基本状態】
kihon.png（自分）のHPは1000　エナジーは10（MAX、初期状態）（自分＝kihon.pngや攻撃時に表示するpngも含む（HPは共有））
damager.pngのHPは5000　自己回復をすることができ、１秒でその時のHPの3パーセントを回復する。発動回数は１０回（HPがマックスの時はカウントしない）
【ダメージなど】当たるとHPが削れる
打撃パンチ：100、気弾：200、波動：500
敵の攻撃（ball.png）：200

【アタック】
敵の攻撃：ball.pngが敵の座標からランダムの時間、ランダムな個数kihon.pngめがけて飛んでくる。
（最短時間間隔は５秒、上限個数は５つ（まれ））　ball.pngは気弾に当たると消える。

手がグーの状態で肘を伸ばすと打撃パンチが打てる。この時kihon.pngからrightattack.png (右手でパンチ)もしくはleftattack.png（左手でパンチ）に変更。肘が曲がった状態から伸びた状態にならないと反映しない。変更したpngの維持時間は約1秒。パンチを打つと、敵の座標にdagekidm.pngを表示して、ダメージを与える。0.2秒間表示する。エナジーを１消費する

片手がパーで腕を伸ばすと気弾を放つことができる。この時kihon.pngからkikouha.pngに変更、肘が曲がった状態から伸びた状態にならないと反映しない。維持時間は約1秒。気弾を放つ動作を行ったときに、kidan1.png、kidan2.png、kidan3.pngを順番に右方向に表示して、気弾が飛んでいるように見せる。敵に当たったら、まとめて消えて、同時に敵の座標にkidandm.pngを0.2秒間表示して、ダメージを与える。エナジーを２消費する

両手がパーで腕を伸ばすと波動を放つことができる。この時kihon.pngからhissatuhado.pngに変更。肘が曲がった状態から伸びた状態にならないと反映しない。維持時間は約1.5秒。波動を放った時hado1.png,hado2.png,hado3.png,hado3.png,hado3.png,hado4.pngを順番に右方向に表示して、波動が飛んでいるように見せる。敵に当たったらまとめて消えて、同時に敵の座標にhadodm.pngを0.2秒間表示して、ダメージを与える。エナジーを5消費する

【防御】
両肘を曲げる（肘が90度以上曲がった時）とkihon.pngの座標のちょっと右よりにgard.pngを追加する。５秒間維持するとHPを100回復できる。さらにエナジーが１プラスできる。ball.pngがgard.pngに当たった時ball.pngは消える。両肘が90度以上曲がっていないときはkihon.pngを表示。
【ゲーム終了判定】
kihon.pngのHPがなくなるか敵のHPがなくなるかしたらゲーム終了、自分が勝ったらサクセス画面。敵が勝ったらゲームオーバー
